A (non-complete) list of features implemented so far
====================================================

* Image Repository: Delivers images for requested size and quality from cache or calculated on the fly
* Remote Interface for Export plugins (not finished yet)
* Ajax-based administration interface featuring
   * Drag & Drop sorting of images
   * Editing image title and description
   * Settings album thumbs
   * Deleting items
   * Editing album title and description
* Import API with implemented uploaders / importers
   * ZIP uploader
   * Directory importer
   * Lightroom importer
* Exif Data parser
* Theming system: implement your own theme easily
* Cooliris Feed (not working at the moment)
